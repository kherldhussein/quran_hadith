# Changelog

All notable changes to this project will be documented in this file.

## 2.0.0

Highlights:
- Hadith: New in-book content widget with infinite scroll, search, and actions (copy/share/save). Book pages now display content reliably.
- Audio: Live speed control, autoplay-next, and repeat modes wired into playback flow; resilient audio URL fallback using global ayah index.
- Desktop: Guarded system tray and notifications to avoid startup crashes; improved Linux build stability.
- About: Modern, theme-aware About dialog with version/build info, quick links, and system details.

Improvements & fixes:
- Offline badges and per-surah download indicators; smoother loading states.
- Hardened reciter parsing and sanitization; minor UI polish and adaptive layout tweaks.
- Dependency updates and missing packages added.

Notes:
- No breaking changes to user data or settings.
- After updating, run `flutter pub get`.
